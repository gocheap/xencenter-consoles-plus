About

Part personal science project, part contribution to the XenServer community, XenCenter Consoles+ is a plugin for Citrix XenServer's management console, XenCenter. For those familiar with the XenCenter Plugins section of the XenServer CDN site, think of XenCenter Consoles+ as a collection of the most handy web consoles and, eventually, the best PowerShell scripts.

Features:
  • Currently supported web management consoles:
    • Citrix Licensing
    • Merchandising Server
    • NetScaler VPX
    • Branch Repeater VPX
    • Access Gateway VPX
    • XenClient VM Synchronizer
    • XenServer Distributed Virtual Switch Controller
    • EdgeSight Server
    • Desktop Director
    • NetApp FilerView
  • Future Plans include useful PowerShell scripts and more web consoles.
  • Community requests welcome!

Usage

In order to get started using XenCenter Consoles+, you'll need to follow these instructions.

  • Enabling and disabling XenCenter Consoles+ is done through the Tools -> Plugins menu.
  • Turning the web console tab on/off is handled through tags on your VMs.
    Web Console Tag
    Access Gateway VPX agvpx
    Repeater VPX brvpx
    DVS Controller dvsc
    EdgeSight edgesight
    Citrix Licensing lic
    Merchandising Server merch
    NetScaler VPX nsvpx
    XenClient VM Synchronizer xcsync
    Desktop Director desktopdirector
    NetApp FilerView netapp
    Vyatta vyatta
  • After enabling the web console tab you need to create a new custom field called 'xcpURL' and fill it in with the URL to load. This provides support for HTTPS, FQDNs, etc.

    • NOTE: FilerView will also be shown on all Storage Repositories utilizing the Direct NetApp Storage Adapter.
